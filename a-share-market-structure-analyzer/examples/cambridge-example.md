# Cambridge Technology Example

Use for a high-position disagreement structure when visible data resembles:

```text
股票：剑桥科技 603083
阶段特征：高位放量大跌，跌破MA5/MA10，MACD死叉绿柱，OBV高位回落但未崩。
筹码：仍在70%成本区内；平均成本约191.70；70%成本区约180.00—201.60；90%成本区上沿约209.70；前高219.49。
```

Answer pattern:

```text
结论：高位分歧，短期偏空，中期趋势仍在关键防守测试中。

核心矛盾：
趋势尚未彻底破坏，但高位放量大跌、MACD死叉绿柱、跌破MA5/MA10，说明短线进攻明显退潮。

关键价位：
- 核心防守区：180.00—185.75
- 修复压力区：191.70—194.33
- 强压力区：201.60—209.70，前高219.49
- 失效条件：跌破180.00且OBV明显下拐，视为高位派发风险升级

操作纪律：
守住180.00—185.75并站回194附近，才叫初步修复；反弹站不上191.70—194.33，偏弱反抽；跌破180且OBV下拐，切换为高位派发风险。
```
