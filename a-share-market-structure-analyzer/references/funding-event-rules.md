# Funding And Event Rules

Events must be interpreted with price position, trend, chips, and volume. Never treat an event alone as decisive.

## Margin Financing

| State | Meaning |
| --- | --- |
| Financing buying increases + price reclaims key level | Leverage may be pushing repair |
| Financing balance high percentile + price breaks down | Leverage reversal risk |
| Margin balance decreases + price falls | Leverage withdrawal |
| Short balance low | Short pressure limited |
| Financing balance above historical 90th percentile | Popular but crowded |

Fixed principle:

```text
融资买入不是单独利好。融资盘在上涨时助涨，在破位时助跌。
```

## Dragon Tiger List

| Structure | Judgment |
| --- | --- |
| Institutions continuously net buy | Relatively strong |
| Institutions sell heavily | Relatively weak |
| Balanced active-money relay | Short-term active |
| One dominant buyer/seller | Dumping or control risk |
| Frequent quant seats | Volatility may expand |
| Buy/sell seats churn each other | Large disagreement |

Prefer institution continuity over one-day appearance. A one-day board cannot override broken trend/chip structure.

## Announcements

| Announcement | General Impact |
| --- | --- |
| Dividend | Neutral with sentiment support |
| Rights issue | Neutral to negative refinancing pressure |
| Private placement | Depends on price, target, and use |
| Buyback | Sentiment positive |
| Earnings pre-increase | Depends on surprise level |
| Shareholder reduction | Negative |
| Contract/order win | Depends on amount and profit elasticity |

Fixed principles:

- `公告利好但股价涨不动，可能是借利好出货。`
- `公告利空但股价不跌，可能是利空出尽或已有消化。`

## Sector Context

- Sector strength can improve repair probability only if the individual stock reclaims key levels.
- Sector divergence matters: strong sector + weak stock means relative weakness.
- Theme acceleration can create chasing risk if individual stock is far above MA20 and chip upper edge.
- Sector rotation failure should make high-position structures more defensive.
